## Compatibility

Babel ensures full ES6 compatibility with all browsers

## Installation

```sh
npm install
```

## Development

```sh
npm start
```

```sh
npm run watch
```

* Browse to ``http://localhost:3000``
* Edit JS/CSS
* Refresh

## Production

```sh
npm run build -- --no-debug
```

Deploy.
